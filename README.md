## Scheduled Messages Cron
It's a cron to send the scheduled messages

### Installation and local launch

1. Clone the repo: `git clone git@github.com:ana-ac/scheduled-messages-cron.git`  
3. Create `.env` with the environment variables listed in the `.env.example`  
4. Run `npm install` in the root folder
